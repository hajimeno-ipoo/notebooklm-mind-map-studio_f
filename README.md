# NotebookLM Mind Map Studio

NotebookLM（Google）のマインドマップ機能を拡張し、デザインのカスタマイズ、レイアウト変更、高解像度画像（PNG/SVG）としての保存を可能にするChrome拡張機能です。

## 主な機能

*   **デザインカスタマイズ**: ノードの色、形状、接続線のスタイル、フォントサイズなどを自由に調整可能。
*   **レイアウト変更**: ツリー型、放射状、フィッシュボーン、年表など多様なレイアウトに対応。
*   **データ取得**: NotebookLM上のテキスト選択や、既存のグラフ構造から自動的にマインドマップデータを生成。
*   **エクスポート**: 作成したマップをPNGまたはSVG形式でダウンロード。
*   **JSON/Markdown編集**: マップの構造をテキストベースで直接編集可能。

## 前提条件

*   Node.js (v18以上推奨)
*   npm (通常Node.jsに同梱されています)

## セットアップ

プロジェクトのルートディレクトリで以下のコマンドを実行し、依存関係をインストールします。

```bash
npm install
```


## ビルド方法

Chrome拡張機能として読み込める形式（`dist` フォルダ）にビルドします。

### 開発中（ホットリロード）
コードを変更すると自動的に再ビルドされます。

```bash
npm run watch
```

### 本番用ビルド
最終的なリリース用に最適化されたビルドを行います。

```bash
npm run build
```
## 一括セットアップ:
依存関係のインストールとビルドを一度に行いたい場合は、以下のコマンドを使えるようにしました。

```bash
npm run setup
```

コマンド実行後、プロジェクトルートに `dist` フォルダが生成されます。この中身が拡張機能の実体です。

## Chromeへのインストール方法

1.  Google Chromeを開き、アドレスバーに `chrome://extensions` と入力して移動します。
2.  画面右上の **「デベロッパーモード」** をオンにします。
3.  左上に表示される **「パッケージ化されていない拡張機能を読み込む」** ボタンをクリックします。
4.  先ほどビルドで生成された **`dist` フォルダ** を選択します。
5.  拡張機能一覧に「NotebookLM Mind Map Studio」が表示されればインストール完了です。

## 使い方

1.  **NotebookLMを開く**: [NotebookLM](https://notebooklm.google.com/) にアクセスし、ノートブックを開きます。
2.  **マインドマップ化**:
    *   **拡張機能サイドパネル**: アイコンから拡張機能のサイドパネルを開き利用。
    *   **フローティングボタン**: 画面上にグラフ要素がある場合、右下に「✨ Studioで開く」ボタンが表示されるのでクリックします。
3.  **編集と保存**: 別タブでStudioが開き、スタイルの変更や画像の保存が行えます。

## トラブルシューティング

*   **「拡張機能が破損しています」と表示される場合**:
    *   `index.html` 内に `<script type="importmap">` が含まれているとエラーになることがあります。`npm run build` を実行することで、ビルドプロセスがこれを自動的に削除・修正します。必ず `npm run build` 後の `dist` フォルダを使用してください。
*   **変更が反映されない場合**:
    *   `chrome://extensions` ページで、該当する拡張機能の「更新」アイコン（回転矢印）をクリックしてリロードしてください。
